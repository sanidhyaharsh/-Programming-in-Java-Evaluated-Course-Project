Student Management System (Java)

Overview:
This is a console-based Student Management System built using Java. It allows users to manage student records efficiently using object-oriented programming concepts.

Features:
* Add student
* View all students
* Search student by ID
* Delete student
* Save data to file
* Load data from file

Concepts Used:
* Classes & Objects
* Inheritance
* Exception Handling
* File I/O Streams
* Collections (ArrayList)

How to Run:
1. Compile:
   javac *.java
2. Run:
   java Main

Project Structure:
* Main.java
* Student.java
* StudentManager.java
* FileHandler.java

Output:
Console-based menu-driven system
